# Data Science Learning Archive

데이터사이언티스트 과정 중 실습한 Jupyter Notebook을 모아둔 **Learning Archive**입니다. 이 레포는 완성형 포트폴리오가 아니라, 통계, 머신러닝, 시계열, 이상거래 탐지, 시각화 등을 학습하며 남긴 기록입니다.

## How To Read This Repository

- 대표 포트폴리오 프로젝트는 아닙니다.
- 과정 중 배운 개념을 실습한 notebook collection입니다.
- 학습 흐름을 보여주기 위한 archive로 남겨둡니다.
- 향후 일부 notebook은 별도 프로젝트 레포로 분리하거나 정리할 수 있습니다.
